## Tech Stack

**Client:** Typescript, React js, Next js, TailwindCSS, Scss




## Run Locally

Clone the project

```bash
  git clone git@github.com:mohaymenrafi/personal-portfolio.git
```

Go to the project directory

```bash
  cd personal-portfolio
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
